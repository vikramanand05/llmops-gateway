# Code of Conduct

## Our Pledge

We as contributors, maintainers, and community members pledge to make participation in this project a respectful, inclusive, and harassment-free experience for everyone.

We are committed to fostering an open, welcoming, diverse, and collaborative environment.

## Expected Behavior

Examples of behavior that contributes to a positive environment include:

* Being respectful and considerate of differing viewpoints and experiences.
* Providing constructive feedback.
* Accepting feedback gracefully.
* Focusing on what is best for the community and project.
* Showing empathy toward other contributors.

## Unacceptable Behavior

Examples of unacceptable behavior include:

* Harassment, discrimination, or personal attacks.
* Offensive comments related to race, ethnicity, gender, religion, nationality, disability, or other protected characteristics.
* Trolling, insulting, or derogatory remarks.
* Publishing private information without permission.
* Any conduct that could reasonably be considered inappropriate in a professional environment.

## Enforcement

Project maintainers are responsible for clarifying and enforcing this Code of Conduct.

Maintainers may remove, edit, or reject comments, commits, code, issues, or pull requests that do not align with this Code of Conduct.

## Reporting Issues

If you experience or witness unacceptable behavior, please contact the project maintainer through GitHub Issues or GitHub Discussions.

All reports will be reviewed and investigated appropriately.

## Scope

This Code of Conduct applies to all project spaces, including:

* GitHub Issues
* Pull Requests
* Discussions
* Documentation
* Community interactions related to this project
